# Vibe Version

This folder contains the task manager app built using a vibe coding tool.

**Tool used:** v0
**Time to generate:** 1 min 42 sec
**Prompt used:** Build a simple personal task manager web app.

                    Requirements:
                    - Add a task with a title
                    - Mark a task as complete
                    - Filter tasks by status: All, Active, Completed

                    UI:
                    - Clean and minimal design
                    - Input field with add button
                    - Task list with checkbox for completion
                    - Filter buttons at the top

                    Technical:
                    - Use React with functional components
                    - Use simple state management (useState)
                    - No backend, store tasks in local state
                    - Keep code modular but simple

                    Do not over-engineer. Keep components small and readable.

**Live URL:** [Link](https://task-manager-frontend-on-vercel.vercel.app/)

## Notes

The tool generated a complete application with a structured file system and working UI in a very short time. However, some components were larger than expected and required effort to fully understand.
